Repositório criado no evento Ignite Lab sobre Design System

## configuração

clonar esse repositório e entrar na pasta criada

```Shell
npm -i # instalação das dependências
```

### Rodando o projeto local

```Shell
npm run dev
```

### Rodando o Storybook

```Shell
npm run storybook
```
